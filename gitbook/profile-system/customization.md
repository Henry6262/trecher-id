# Customization

Make your profile yours with custom branding options.

## Accent Color

Choose a custom accent color that changes the entire profile's look:
- Header gradient
- Badge and tag borders
- Link hover effects
- Chart colors
- Glow effects

Pick any hex color from the dashboard color picker. Default is cyan (#00D4FF).

## Banner Image

Add a custom banner image to the top of your profile:
- Displays above the profile header
- Dark gradient overlay ensures text readability
- Auto-fetched from Twitter banner (via fxtwitter) if available
- Can be overridden with a custom URL

## Display Name & Bio

- **Display name** — shown prominently on your profile
- **Bio** — short description (shown below your name)
- Both editable from the dashboard

## Social Links

Add links to your other platforms:
- Twitter
- Telegram
- YouTube
- Discord
- Twitch
- Custom URLs

Links display with platform icons and the Web3Me skewed container styling.

## Design Language

All customization respects the Web3Me design system:
- **Cut-corner clip paths** — no border-radius, only angular cuts
- **Glassmorphism** — frosted glass backgrounds with blur
- **Geist Sans** (body) + **Geist Mono** (numbers/addresses)
- **Dark theme only** — background #050508
