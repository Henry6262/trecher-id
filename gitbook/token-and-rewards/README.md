# $WEB3ME Token

$WEB3ME is the native token of the Web3Me platform, launched on Solana.

## Purpose

The token serves one critical function: **funding the Reward Vault** that powers the Trencher Cup and weekly leaderboard payouts.

Every $WEB3ME transaction includes a creator fee. **69% of that fee flows directly into the Reward Vault.** This creates a direct link between token activity and the prize pool available to top traders.

## Fee Structure

```
Every $WEB3ME transaction
        │
        ↓
   Creator Fee (X%)
        │
        ├── 69% → Reward Vault
        │         ├── Trencher Cup seasonal prizes
        │         └── Weekly Top 10 leaderboard payouts
        │
        └── 31% → Platform operations & development
```

## Why 69%

This isn't a meme number (okay, maybe a little). The 69% allocation ensures:

1. **Majority of revenue goes to traders** — the people who make the platform valuable
2. **Sustainable development budget** — 31% covers servers, APIs, and continued feature development
3. **Aligned incentives** — platform succeeds only if traders succeed
